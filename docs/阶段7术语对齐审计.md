# 阶段 7 术语对齐审计

审计依据：

- `00_合并执行路线图.md` 阶段 7。
- `04_跨教材整合与压缩阶段计划.md` 中的候选召回和相似度原则。
- `11_数据清洗与知识图谱关键问题清单.md` B/C/D/G。

## 当前目标

阶段 7 只判断“是不是同一概念或别名候选”，不做压缩和删除。真正 `merge/keep/remove/refine/conflict` 决策留到阶段 8。

## 当前实现是否达到阶段 7 要求

已达到：

- 能加载 2 本以上已构建图谱。
- 能生成跨教材 `AlignmentCandidate`。
- 能把同义/别名节点聚成 `ConceptCluster`。
- 每个 cluster 有 `canonical_name`、`aliases`、`member_node_ids`、`confidence`。
- 每个候选有 `signals`、`reason`、`source_locators`、`evidence_chunk_ids`。
- 低置信或冲突候选会标记 `needs_teacher_review`。

不是只看“有没有”的质量断言：

- `smoke_00_stage7_alignment.py` 断言白细胞/白血球/leukocyte 能进入同一 cluster。
- `benchmark_00_stage7_alignment.py` 断言 3 个期望同义簇 recall、禁合并对、候选信号完整度。
- alias 提取只接受绑定到当前节点名的显式别名，例如 `白血球又称 leukocyte`，避免把同一句里的普通片段误当 alias。

## 对齐信号

每个候选由以下信号组成：

- `normalized_name`：名称规范化后是否一致。
- `alias_table`：内置 alias/canonical 表或显式别名是否命中。
- `definition_similarity`：定义 token Jaccard 相似度。
- `context_similarity`：章节上下文 token 相似度。
- `neighbor_similarity`：图谱邻居 canonical key 相似度。
- `hash_embedding`：本地 hash embedding 余弦相似度。

## 当前边界

当前不会：

- 直接合并节点。
- 删除低价值节点。
- 计算压缩比。
- 为冲突做最终裁决。
- 覆盖教师决策。

这些属于阶段 8 和阶段 10。

## Benchmark 当前标准

`benchmark_00_stage7_alignment.py` 输出：

- `cluster_recall`：期望同义簇召回率。
- `forbidden_pair_hit_count`：禁合并对误聚类次数。
- `candidate_signal_completeness`：候选是否都有信号、理由、证据。
- `average_build_ms`：构建耗时。

当前目标线：

- `cluster_recall >= 1.0`。
- `forbidden_pair_hit_count == 0`。
- `candidate_signal_completeness == 1.0`。

最近一次回归结果：

- `cluster_recall = 1.0`。
- `forbidden_pair_hit_count = 0`。
- `candidate_signal_completeness = 1.0`。
- `expected_cluster_count = 3`。
