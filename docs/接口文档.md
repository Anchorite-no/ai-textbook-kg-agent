# 接口文档

本项目以后端 Pydantic schema 和 FastAPI OpenAPI 为契约事实来源。

## 契约来源

- 在线契约：`GET /openapi.json`
- 离线快照：`docs/openapi.snapshot.json`
- 数据结构说明：`docs/数据契约.md`
- 前后端对接清单：`docs/后端接口对接清单.md`

## 核心接口

| 模块 | 接口 | 用途 |
| --- | --- | --- |
| 健康检查 | `GET /api/health` | 检查后端服务状态 |
| 数据集 | `GET /api/datasets/seven-books` | 读取七本教材 demo 数据集状态 |
| 数据集 | `POST /api/datasets/seven-books/prepare` | 准备七本教材 demo 数据 |
| 上传整理 | `POST /api/workflows/organize` | 前端上传一个或多个文件并触发解析、KG、RAG、整合流程 |
| 教材 | `GET /api/textbooks` | 列出已解析教材 |
| 教材 | `GET /api/textbooks/{raw_file_id}` | 查看单本教材结构和 chunk |
| 知识图谱 | `POST /api/graph/build` | 构建单本教材 KG |
| 知识图谱 | `GET /api/graph` | 读取图谱节点、边和统计 |
| 多层 KG | `POST /api/kg/layers/build` | 构建 document tree、concept KG、evidence graph |
| RAG | `POST /api/rag/index` | 构建 chunk 检索索引 |
| RAG | `POST /api/rag/query` | 返回带 citation 的精准问答结果 |
| GraphRAG | `POST /api/graphrag/query` | 返回答案、引用、节点命中、路径和整合决策 |
| 对齐 | `POST /api/alignment/build` | 跨教材术语和概念对齐 |
| 对齐 | `GET /api/alignment` | 读取对齐候选和 cluster |
| 整合 | `POST /api/integration/build` | 生成跨教材压缩整合决策 |
| 整合 | `GET /api/integration` | 读取整合结果 |
| 教师修改 | `POST /api/integration/decisions/{decision_id}/override` | 教师覆盖整合决策 |
| 对话 | `POST /api/dialogue/messages` | 教师自然语言反馈入口 |
| 报告 | `POST /api/report/generate` | 基于真实后端产物生成 Markdown 报告 |

## 错误格式

所有业务错误统一返回：

```json
{
  "message": "错误说明",
  "code": "ERROR_CODE",
  "detail": {}
}
```

## 前端同步

修改后端契约时，需要同步：

1. `backend/app/models/schemas.py`
2. `docs/数据契约.md`
3. `docs/openapi.snapshot.json`
4. `frontend/src/types/api.generated.ts`

导出命令：

```powershell
cd D:\Hackathon
$env:PYTHONPATH=(Resolve-Path backend).Path
backend\.venv\Scripts\python.exe backend\scripts\export_openapi.py
```
